Dicoding Course

This is the description of my final project on Dicoding course. The project focused on building a responsive web application using the foundational web technologies: HTML, CSS, and JavaScript. For structuring the content, I applied semantic HTML elements like header, main, footer, article, and aside to ensure better accessibility and SEO. The design and layout were handled using modern CSS features, including Flexbox to create a flexible, responsive layout that adapts to different screen sizes.

To enhance user experience, I also implemented basic interactivity using JavaScript. This includes a mobile-friendly navigation menu with a toggle button, which dynamically shows or hides the menu when clicked. The layout was designed with mobile-first principles, ensuring the website looks great on smaller screens, and scales smoothly on larger displays.

Additionally, I paid special attention to design aesthetics by using a consistent color scheme and typography. The project utilized external stylesheets for CSS, making the codebase clean and maintainable. This project not only demonstrated my understanding of web development fundamentals but also my ability to integrate responsive design and enhance usability for all types of users.
